# Twitter Demo Script

## Setup
- Clean terminal, dark theme
- Have Claude Code open
- Screen recording at 1080p or 4K

## Recording Script (45 seconds)

### Scene 1: Hook (5s)
*Show terminal with Claude Code*
Type: `# Let's make Claude Code talk`

### Scene 2: Basic Speak (8s)
Type: `/speak Hello! I can talk now.`
*Wait for audio to play*

### Scene 3: Emotions (15s)
Type: `/speak [excited] The tests all passed!`
*Wait for excited voice*

Type: `/speak [whispers] But there's one more secret feature...`
*Wait for whisper*

Type: `/speak [laughs] Just kidding, we're done!`
*Wait for laugh*

### Scene 4: Notification Demo (10s)
*Show Claude working on something*
*Walk away from keyboard (or pretend)*
*Audio plays: "Claude has finished responding"*
*Come back*

### Scene 5: Call to Action (7s)
Type or show text overlay:
```
github.com/adamanz/voice-elevenlabs
/plugin install adamanz/voice-elevenlabs
```

## Post-Recording
- Speed up any waiting parts (1.5x-2x)
- Add captions for accessibility
- Add background music (optional, low volume)
- Export as MP4 and GIF

## Tools
- Screen recording: QuickTime, OBS, or Screen Studio
- GIF conversion: `ffmpeg -i demo.mp4 -vf "fps=15,scale=720:-1" demo.gif`
